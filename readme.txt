This plugin extend Stella plugin abilities about localization metaboxes.

== Description ==
Plugin requires your theme to use Custom Metabox API https://github.com/WebDevStudios/Custom-Metaboxes-and-Fields-for-WordPress

== Installation ==
1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings->Stella CMB
4. Select metaboxes, which should be localized

Yes, you should install Stella plugin first
